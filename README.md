# [m|g]?vim configuration

Repo holding personal vim files.

## Installation

Backup old settings

    $ mv .vim    .vim.bak   && \
      mv .vimrc  .vimrc.bak && \
      mv .gvimrc .gvimrc.bak

Install

    $ git clone http://github.com/torgeir/vim.git && \
      ln -s vim .vim                              && \
      ln -s vim/.vimrc                            && \
      ln -s vim/.gvimrc

Done!
